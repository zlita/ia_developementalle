Nous avons crée une classe Agent comportant 2 actions,une liste d'action en mémoire et une fonction de prédiction.
Nous avons aussi une classe Env comportant deux environnement et une fonction feedback.
Notre agent choisi un action, appel la fonction de prédiction.
Il obtient un feedback par rapport à son action. Si la prédiction est la même que le feedback alors notre agent est content.
Sinon il n'est pas content et remplace dans sa liste "mémoire" l'action correspondante.
